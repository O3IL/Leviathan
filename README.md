## Features:
* Compact Overlay mode will make the app stay on top of other apps or windows.
* Focus mode hides all distractions.
* Classic Mode.
* Auto Save.
* Dark Mode.
* Supports rich text formats.

## Planned features:
* File Print.
* Tabs.
